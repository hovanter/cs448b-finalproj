Data tags Detail
Ordinal An ordinal value has some kind of ordering defined over it, which establishes a relative ranking over the data points. For example, days of the week (Monday, Tuesday, etc...) have an order defined over them.
Nominal A categorical value represents a category. It has no intrinsic ordering. For example, colors are a categorical value. Red, blue, and yellow each define a type of color, but there's no intrinsic reason why red is more important than blue, or blue more so than yellow.
Temporal A temporal value is some measure of time. For example, minutes, years, and months are a temporal value.
Quantitative A quantitative measure represents a numerical value.
